# St. Olan Isle Wine Label Presentation

A sophisticated, 4A advertising agency-style presentation website for St. Olan Isle (圣澳蓝屿) wine label designs.

## Features

- **Password Protection**: Elegant landing page with password authentication (password: "jose")
- **Cinematic Intro**: Stunning introduction page with smooth animations
- **Design Overview**: Gallery showcasing all 5 wine label design styles
- **Detailed Views**: Immersive full-screen presentations for each design
- **Smooth Transitions**: Premium animations using Framer Motion
- **Bilingual**: English and Chinese content throughout
- **Responsive**: Optimized for desktop presentation viewing

## Design Styles

1. **圣十字 (Holy Cross)** - Sacred elegance with dual-tone design
2. **复古航海徽章派 (Vintage Nautical Badge)** - Classic maritime tradition
3. **奢华白底系列 (Luxury White Series)** - Modern luxury with clean aesthetics
4. **十字军战士系列 (Crusader Warrior)** - Heroic historical design
5. **水彩奢华海洋宝藏系列 (Watercolor Ocean Treasure)** - Artistic watercolor and gold

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
npm start
```

## Tech Stack

- **Next.js 15** - React framework
- **TypeScript** - Type safety
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **React 18** - UI library
